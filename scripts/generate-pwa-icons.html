PWA Icon Generator

Generate all required PWA icons for SuiDex Games

Instructions:

  1. Click "Generate All Icons" to create icons
  2. Right-click each icon and "Save Image As..." to download
  3. Save them to public/icons/ folder
  4. Or use "Download All" (requires JSZip library)

Required files:

  • icon-72.png, icon-96.png, icon-128.png, icon-144.png
  • icon-152.png, icon-192.png, icon-384.png, icon-512.png
  • badge-72.png (for notifications)
